Cipher

Freestyle Rap. Reimagined.

Cipher is an exciting app that serves 10 second long "beat bites" to users and has them record a verse, whether it be a rap, spoken word, or a poem. Then, it stiches various bites together to create a song, that users can then play back and share. Current implementation is ad-hoc (local) only, but a back-end and complete refresh is in the works.

Check out the code on GitHub.

Unplug

Sometimes you need to get it away from it all.

Unplug harnesses the power of the iPhone's accelerometer and gyrometry systems to encourage users to step away from their phones every now and then. Once a user starts the game, Unplug will detect touch and changes in the phone's linear acceleration or orientation in space (both indicators of someone using the phone) and will automatically send a game over alert. Last as long as you can without your phone, and then post your score to social networks!

Check out the code on GitHub.

Tipsta

Tipping doesn't have to be boring

What started out as an experiment in UI design became Tipsta. One of the easiest to use and minimal tip calculators for the iOS platform. Retains all feautures of a tip calculator (split, tip percentage) and enables easy access to major social networks, so that you can eat, pay, and share, all from the same app.

Check out the code on GitHub.
